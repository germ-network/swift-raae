# New Package template with changesets

Instructions
- Create a new repo using this template
- fill in the repo in `.changeset/config.json`
- fill in the name in `package.json`
- fill in the schema name in `.github/workflows/ci.yml`
- grant [Changeset Bot](https://github.com/apps/changeset-bot) access to your cloned repo
- Replace this readme
- Add a license if desired
